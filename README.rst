diysystem
=========

Do It Yourself Raspberry Pi server status monitoring class. 

Usage
-----

`import diysystem`
`SYSTEM = diysystem.DiySystem(CLIENT, HOSTNAME, INTERVAL)`


Installation
------------


 git clone https://github.com/parttimehacker/Python-DIY-System.git::
  python setup.py install
